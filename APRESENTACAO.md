# 🚀 FinBot V6 — Apresentação Comercial

## O que é o FinBot?

Um **assistente financeiro pessoal via Telegram** que transforma a forma como você controla seus gastos. Simples como mandar uma mensagem, poderoso como uma planilha inteligente.

---

## 🎯 Para quem é?

- Quem quer **controlar gastos** sem abrir planilhas
- Quem quer **saber para onde vai o dinheiro** todo mês
- Quem quer **economizar** com inteligência
- Quem quer um **dashboard bonito** para visualizar tudo

---

## ✨ Funcionalidades

### 💸 Registro de Gastos
- **Modo rápido**: `/g 50 mercado pix` → pronto!
- **Modo guiado**: passo a passo com botões
- **Categorização inteligente**: o bot categoriza automaticamente
- **Assinaturas**: detecta gastos recorrentes (Netflix, Spotify...)
- **Parcelamento**: pergunta se é parcelado e registra automaticamente

### 📊 Dashboard Interativo
- Gráficos de pizza (por categoria)
- Gráficos de barras (por dia)
- Cards com métricas (total, saldo, maior gasto)
- Acessível pelo navegador

### 📈 Insights Automáticos
- "Você gastou 40% mais em delivery essa semana"
- "Seus gastos com moto aumentaram 25%"
- "Economia do mês: R$ 500"
- Detecta padrões e sugere economias

### 🛒 Lista de Compras
- Categorias: mercado, online, farmácia, casa
- Quantidade: `/compras add leite 2x`
- Marcar como comprado: `/compras comprado 1`
- Lista de desejos para itens futuros

### 👤 Perfil Configurável
- Alterar renda: `/perfil renda 3500`
- Alterar nome: `/perfil nome Caio`
- Alterar cartões: `/perfil cards Nubank, Itau`

### 💵 Renda Extra
- Registrar: `/receita 500 freela`
- Controla separadamente na planilha

## 📱 Como usar

1. Inicie com `/start`
2. Use os botões do menu
3. Registre gastos com `/gasto` ou `/g`
4. Veja insights com `/insights`
5. Acesse o dashboard pelo link do relatório
